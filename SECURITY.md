# Security Policy

## Supported Versions

Currently, only the latest release of Zodify JSON receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 0.0.1| :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within Zodify JSON, please send an e-mail to the maintainer rather than opening a public issue. We will review the report and respond as quickly as possible.
