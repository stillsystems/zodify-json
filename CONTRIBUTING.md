# Contributing to Zodify JSON

Thank you for your interest in contributing to Zodify JSON!

## Local Development

1. **Prerequisites**: Ensure you have Node.js v24+ (matching `.nvmrc`) installed.
2. **Install Dependencies**: Run `npm install`.
3. **Run the Extension**: Open the project in VS Code and press `F5`. This will launch an Extension Development Host with Zodify JSON loaded.
4. **Run Tests**: You can run the test suite locally using `npm run test`.
5. **Code Style**: We use ESLint. Please ensure `npm run lint` passes before submitting a pull request.

## Pull Requests

1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Add or update tests as necessary.
4. Submit your pull request against the `main` branch.
